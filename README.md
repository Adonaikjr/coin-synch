## Para iniciar o projeto:
- yarn version: 1.22.19 || npm version: 9.8.0
- node version: 18.16.1
- Clone o projeto
- cd coin-synch
- yarn || npm i
- yarn dev || npm run dev
- Para iniciar fake backend: json-server --watch db.json, certifique-se de ter installado.
